# Screens

All breakpoints using on Design System.

| Size  | Variable              | Device        |
| ----- | --------------------- | ------------- |
| 36rem | `theme.breakpoint.sm` | Mobile        |
| 48rem | `theme.breakpoint.md` | Tablet        |
| 62rem | `theme.breakpoint.lg` | Desktop       |
| 75rem | `theme.breakpoint.xl` | Large Desktop |

We also has a variable to define the max size of containers.

| Size   | Variable             | Device        |
| ------ | -------------------- | ------------- |
| 54rem  | `theme.container.sm` | Mobile        |
| 72rem  | `theme.container.md` | Tablet        |
| 96rem  | `theme.container.lg` | Desktop       |
| 114rem | `theme.container.xl` | Large Desktop |
